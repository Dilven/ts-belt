title: Installation
---

Use either `npm` or `yarn` to install the package:

```shell
yarn add @mobily/ts-belt
```

```shell
npm install @mobily/ts-belt --save
```

### TS Config

Update `strictNullChecks` and `strict` in your `tsconfig.json` (optional, recommended):

```json
{
  "compilerOptions": {
    â€¦
    "strict": true,
    "strictNullChecks": true
  }
}
```
